Acerca

BarberFlow para Vercel y Supabase

Proyecto académico con frontend estático, Vercel Functions y Supabase Postgres.

Objetivo

Gestionar citas reales de barbería sin PHP ni almacenamiento JSON.

APIs

Nager.Date para feriados, Open-Meteo para clima y QRServer para comprobantes.

Backend

Node.js en Vercel Functions conectado a Supabase con service role key solo en servidor.
